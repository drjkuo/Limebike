# LimeBike -- Bike Basket

## Install

1. Install prerequisites:

   * [NPM](https://www.npmjs.com/)
   * [Node](https://nodejs.org/en/)

2. Install this repo:

    `npm install`

## Demo

run `node index.js`

![alt](https://github.com/drjkuo/Limebike/blob/master/index.png)




## Testing

`npm install -g mocha`

run `mocha`
